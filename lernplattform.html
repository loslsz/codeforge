Die while-Schleife

Lektion 3 / 6 · Python Grundlagen · ⏱ ~8 min

Eine while-Schleife wiederholt einen Block so lange, wie eine Bedingung True ist. Sie ist besonders nützlich, wenn du nicht weißt, wie oft eine Schleife laufen soll.

Die Grundstruktur sieht so aus:

  beispiel.py 
while bedingung:
    # Code der ausgeführt wird
    # solange bedingung True ist

Ein wichtiges Beispiel: Zähle von 1 bis 5:

  zähler.py 
i = 1
while i <= 5:
    print(f"Zahl: {i}")
    i += 1

# Ausgabe:
# Zahl: 1
# Zahl: 2  ... bis 5
⚠️ Achtung – Endlosschleifen!
Vergiss nicht, die Bedingung irgendwann zu ändern, sonst läuft die Schleife ewig. Immer sicherstellen, dass i erhöht wird!
